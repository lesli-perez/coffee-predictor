# Coffee Rating Predictor
A data science project predicting coffee review sentiment using machine learning and natural language processing (NLP). This project was completed as the final project for the Data Mining course (CSCI 6366) at the UTRGV.

---

## How to Use

1. Clone the repository.
2. Install dependencies.
3. Add your datasets (X_train.csv, X_test.csv, y_train.csv) to the project folder.
4. Run the project:
  - Notebook: Open CoffeeRating.ipynb in Jupyter Notebook to view all analysis, visualizations, and step-by-step methodology.
